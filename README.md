# Event Registration System

## 📌 Project Description
This is a simple web application where students can register for events. Users can enter their name and choose an event using a form. The data is stored in Firebase Firestore (cloud database).

---

## 🚀 Features
- Easy to use registration form  
- Stores data in cloud (Firestore)  
- Shows success message after submission  
- Website is publicly accessible  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  
- Firebase Firestore  
- Google Cloud Platform  

---

## ⚙️ How to Run the Project

1. Download or clone the project  
2. Open the project folder  
3. Add your Firebase configuration in the code  
4. Open `index.html` in your browser
